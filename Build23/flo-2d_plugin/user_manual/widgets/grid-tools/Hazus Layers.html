14. Water Surface and Depth for HAZUS

This tool is used to assign water surface and depth to the grid layer.

Load MAXELEV

  1. Click on Layer>Add Layers >Add Delimited Text Layer or click on the Open Data Source Manager button and navigate to the Delimited Text tab.

  1. Add the MAXELEV.OUT, following the figure below.

  1. Check if the data was correctly processed.

Note

For a better visualization of the FLO-2D outputs, use MapCrafter. Check the FLO-2D-MapCrafter Wiki.

Assign Water Elevations/Flow Depths to Grid from Points Layer

  1. Click on the Assign Water Elevations/Flow Depths to Grid from Points Layer button.

  1. Edit the dialog box as shown below and click the Assign to selected grid field button.

  1. Wait the process to finish and click OK.

  1. Check the Grid Attribute Table to verify if the data was correctly assigned.

Note

The same process can be performed for the DEPTH.OUT file.
