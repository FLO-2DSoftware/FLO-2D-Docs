Plugin Features

Plugin Menu

The FLO-2D Plugin menu is in the Plugins drop down menu, Plugins>FLO-2D. If the Plugin is not listed, follow the installation instructions to install the Plugin.

List of Additional Plugins

Other helpful Plugins include:

  • FLO-2D Rasterizor

  • FLO-2D MapCrafter

  • Profile tool

  • Quick Map Services

  • Curve Number Generator

Note

For QuickMapService, it is necessary to install and add maps. Click on the QuickMapServices button > Settings > More Services > Get Contributed Pack> Save.
