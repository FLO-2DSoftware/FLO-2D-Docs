Installation
============

The installation process for our software is straightforward and can be completed in a few simple steps.

Prerequisites
-------------

- FLORunner v1.0.0.7z

Installation Steps
-------------------

1. **Download the Software**: Obtain the FLORunner from the FLO-2D sharefile.
2. **Extract the Files**: Unzip the downloaded file to a desired location on your computer.
3. **Copy all contents to the FLOPRO folder**:
    3.1. Move all extracted files into the FLOPRO directory where FLO-2D is installed.
    3.2. Ensure that the `FLORunner.exe` file and the resources and ui folders are located in the same directory as the FLO-2D executable.
4. **Run the Application**: Double-click on `FLORunner.exe` to launch the application.
5. **Check installation**: The run button must be enabled when you open the application.
