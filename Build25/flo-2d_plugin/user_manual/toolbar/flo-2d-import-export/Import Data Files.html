Import data (*.DAT) files

This tool imports existing FLO-2D *.DAT files into the FLO-2D GeoPackage.

Import the FLO-2D *.DAT files into the Project

Important

Before performing this task, set up a New FLO-2D Project.

  1. Click the Import data (*.DAT) files.

  1. Navigate to the FLO-2D *.DAT files

  2. Select CONT.DAT and click Open.

  3. All FLO-2D files are imported to the Plugin with this option.

  1. Choose the *.DAT files to import and click OK.

  1. Once the project import is complete, this message will appear. Click OK to close the message.

  1. The final step is to convert the new layers to User Layers. This process should be automatic. It uses this tool: Convert Schematic Layers to User Layers. If the conversion does not happen automatically, you can manually convert the layers by clicking on the Convert Schematic Layers to User Layers button.
