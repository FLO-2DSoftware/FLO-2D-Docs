GDS - Mapper Install Instructions

Step 1: Download the GDS-Mapper Installer

FLO-2D Pro Subscribers should log into FLO-2D.ShareFile.com and download the GDS-Mapper installer from their FLO-2D Tech Support Account.

ShareFile Login

Important

GDS and Mapper are old processor programs. It may be challenging to install them on Modern Windows systems. If any installation errors show up, try the Ignore option or see the Troubleshooting section.

  1. Go to FLO-2D Pro Update >> Processor Programs >> GDS Mapper Installer and download the file GDS-Mapper-Setup.exe.

Step 2: Run the Installer

  1. Run the file with Admin Rights.

  1. If the installation fails or GDS/Mapper wont run, there are work around instructions in the Troubleshooting section.

Step 3: Test GDS

  1. Open GDS and open a project.

  1. Load a Project. Barn Mudflow is the fastest test.

  1. Click OK to load the components.

  1. Street Errors??? Click OK or Enter key to close all of the Street error messages. FLO-2D seems OK with the streets but GDS doesn’t like them.

  2. Click the Select grid elements with a polygon button.

  3. Click YES to create a grid layer.

  4. Click 3 vertices to draw a simple polygon and double click to close.

  1. If no error message appears and some grid elements turn green, the test passed.

  2. If error messages appear, review the Troubleshooting Section page on the left sidebar.

Step 4. Run the model

  1. Click File Run FLO-2D.

  2. Select the Run Do Not Save option.

  1. If the Blue Message says PROFESSIONAL LICENSE, the computer is activated.

  2. If it says Basic, see FLO-2D Pro Install Instructions to run the activator.

  1. When the model is finished, close the run.

Step 5. Run the Mapper test

  1. From the GDS File Menu, open Mapper Pro.

  1. From the Mapper File Menu, Read FLO-2D Results…

  1. Load FPLAIN.DAT from the file window.

  1. Load a few maps to test Mapper.

  1. If no error messages appeared, Mapper is intsalled correctly.

  2. If error messages appeared, see the Troublehsooting Section on the left sidebar.
