Culverts

Overview

This tutorial outlines the process of creating hydraulic structures with generalized culvert equations.

Required Data

File

Content

*.shp

Culvert Data

Path: …\Coastal Training\Project Data\Culverts

Step 1: Import data

Start by cleaning up the map space so the next layer will be easy to see.

  1. Uncheck the User Left bank, Right Bank Cross section layers.

  2. Check the Elevation Layer.

  3. Click the Structure Lines layer to activate it. It is in the User Layers Group.

  4. Drag the Culvert Data.shp onto the map space.

Step 2: Format the data layers

  1. Double click the Culvert Data layer

  2. Change the symbology to Arrow.

  3. Set the Labels to structname.

  1. This helps identify the structure and show the flow direction.

Step 3: Build the structures into the User Layers.

  1. Use the Structure Editor to add all of the new structures.

  2. Collapse the Widgets, open the Structures Editor, and click the Add a Structure button.

Digitizing process:

  1. Left click the inlet node (upstream node)

  2. Left click the outlet node (downstream node)

  3. Right Click to finish the polyline. Click OK to close the attribute window.

  4. Watch the gif for a cycle. Notice how the inlet and outlet position try to take advantage of the cells with the lower elevations. The positioning is not perfect. It is aligned to the cell that most closely represents the invert.

Step 4: Assign the structure attributes

  1. Click Save in the Structures Widget and check the center box.

  1. Open the Culvert Data attribute table. The attributes will help fill out the data for each structure.

  1. Use the following Image or the attribute table to fill out the culvert variables.

  2. If the following image is not clear, right click it and open it in a new tab.

  1. For Culvert Equations, extra data is needed in the Table Editor. If the following image is not clear, right click it and open it in a new tab. The new plugin also has a Mult_Barrel option. Set Mutl_Barrel to 1.

  1. Schematize the structure data and click Yes to replace the data.

Step 6: Save, and export

  1. This is a good point to save project.

  1. Export the data files to the Project Folder in QGIS Lesson 6 Export.

  1. All data files will be created in the selected project folder.

Step 7: Create a backup file

  1. Close QGIS.

  2. Open the project folder. Select the Coastal Project.gpkg and Coastal Project.qgz files. Right click them and click Sent to/Compressed (zipped) folder.

  1. Name the zipped file. It is good to choose a name that identifies project progress. For Example: CulvertsOK.zip

  1. Open QGIS and reload the project.

  1. Click yes to load the model.
