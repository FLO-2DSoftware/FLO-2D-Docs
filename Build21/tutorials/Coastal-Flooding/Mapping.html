Step 2. Import results from mesh tool

  1. Click the Data Management button.

  1. Select the Mesh option.

  2. Navigate to the FLO-2D Project folder and select the DEPTH.OUT file.

  3. Click Add and select the 2D layer and click OK.

  1. Drag the New Layer and drop it into the Results Group.

Step 3. Load the Layer Style panel

  1. Dock the layer styling panel with the FLO-2D widgets.

  2. Select the Depth layer in the Results Group.

  3. Drag the edge of the Layer Styling Panel slightly to the left to make it wider.

  4. Click the multi-colored block next to Depth to load Depth from the mesh layer.
