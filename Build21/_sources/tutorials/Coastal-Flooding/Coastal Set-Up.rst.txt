Coastal Set-Up
==============

**Overview**

Please use these instructions to get your computer ready for Coastal
Training Package.

.. important:: Complete FLO-2D Installation & Setup prior to continuing on this training.

Step 1: Get the Data
----------------------

1. Download the data installer.

|Setup_Download|

.. |Setup_Download| raw:: html

   <a href="https://flo-2d.sharefile.com/d-s7b49c33fcc6744adb2c28cc10ad56bfc" target="_blank">Click Here to Download</a>

2. Run the installer.

.. image:: ../img/Coastal/setup009.png

3. Close the installer if a Completed message appears.

.. image:: ../img/Coastal/setup010.png

4. Navigate to the training folder and pin it to Quick Access.

Path: C:\\Users\\Public\\Documents\\FLO-2D PRO Documentation\\Example Projects\\Coastal 2D Training

.. image:: ../img/Coastal/setup011.png


Step 2: Setup QGIS
------------------

.. note:: Setup must be performed by the student not IT Admin as it will define the Student's QGIS Profile.

1. Load QGIS

.. image:: ../img/Coastal/setup001.png

2. Close the Browser Panel or dock it with the Layers Panel. The video
   shows how to dock it.

.. image:: ../img/Coastal/setup002.png


3. Set the CRS options as shown in the following image.

.. image:: ../img/Coastal/setup003.png

Step 3: Add Plugins to QGIS
---------------------------

1. Open the Plugin Manager.

.. image:: ../img/Coastal/setup004.png

2. Add the FLO-2D Plugin v0.10.115. Use Install from Zip option.

.. image:: ../img/Coastal/setup005.png

3. Add all other plugins using the normal plugin installation method
   from the All Group.


.. note:: If the following image is hard to see, open it in a new tab.


.. image:: ../img/Coastal/setup006.png

Step 4: Add Quick Map Services
------------------------------

1. Open the Quick Map Services Settings.

2. Go to the More Services tab and Get Contributed Pack

.. image:: ../img/Coastal/setup007.png

3. Uncheck all visible items and turn on Google, ESRI, and OSM.

.. image:: ../img/Coastal/setup008.png
